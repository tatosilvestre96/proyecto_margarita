# Backlog Command

## Descripción
Sistema para agregar ideas rápidamente al backlog del proyecto.

## Uso
Escribe cualquiera de estas formas:

### Con categoría
```
/backlog backend: Tu idea aquí
/backlog frontend: Tu idea aquí
/backlog video: Tu idea aquí
/backlog bug: Tu idea aquí
```

### Sin categoría (usa Mejoras Generales)
```
/backlog: Tu idea rápida aquí
```

## Categorías disponibles
- `backend` → 🏗️ Backend
- `frontend` → 🎨 Frontend
- `gameplay` → 🗺️ Gameplay
- `video` → 🎬 Video Content
- `database` → 🗄️ Database
- `security` → 🔐 Seguridad & Performance
- `multiplayer` → 📱 Multiplayer
- `bug` → 🐛 Bugs & Fixes
- `docs` → 📚 Documentación

## Ejemplos
```
/backlog backend: Crear endpoint para guardar progreso
/backlog frontend: Mejorar responsive del mapa
/backlog video: Grabar intro del primer nivel
/backlog bug: Socket.io desconexión después de 5 min
/backlog: Se me ocurrió un nombre mejor para el juego
```

## Archivo
Todas las ideas se guardan en: `BACKLOG.md`
